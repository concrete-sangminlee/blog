박사과정인데 이런 것도 매번 검색해? 라는 말을 들을 수도 있지만, 솔직히 적어본다.

## Python 관련

- `numpy reshape order C F` : C order인지 F order인지 매번 헷갈린다
- `matplotlib legend location` : 범례 위치 옵션이 뭐가 있었더라
- `pandas merge left right` : join 방향이 항상 헷갈린다
- `python string format f-string` : 포맷 문법을 자꾸 까먹는다

## Git 관련

- `git rebase vs merge` : 알고 있는데 매번 확인하게 된다
- `git undo last commit` : reset이었나 revert였나
- `git stash pop` : stash 문법이 직관적이지 않다

## LaTeX 관련

- `latex table centering` : 표 가운데 정렬이 왜 이렇게 어려운지
- `latex figure placement h t b` : 그림 위치 옵션이 매번 말을 안 듣는다
- `bibtex vs biblatex` : 차이가 뭐였더라

## 기타

- `how to read a paper efficiently` : 논문 읽는 법을 검색하는 아이러니
- `is it ok to email professor` : 교수님께 메일 보낼 때마다 예의를 검색한다
- `timezone converter` : 해외 학회 일정 확인할 때마다

## 결론

자주 쓰는 것도 외워지지 않는 건 정상이다. 중요한 건 외우는 게 아니라, 빠르게 찾아서 쓸 수 있는 거다. 라고 스스로를 위로한다.
