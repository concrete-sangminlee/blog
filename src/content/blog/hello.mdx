블로그를 하나 만들었다.

연구하면서 배운 것들, 삽질한 것들을 어딘가에 기록해두고 싶다는 생각은 예전부터 있었는데, 매번 "나중에 해야지"만 반복하다가 결국 이번에 만들었다.

재밌는 건 이 블로그를 Claude랑 같이 만들었다는 거다. Astro라는 프레임워크로 정적 블로그를 세팅하고, 디자인 잡고, GitHub Pages에 배포하는 것까지 대화하면서 진행했다. 직접 하나하나 다 짰으면 꽤 오래 걸렸을 텐데, 대화하면서 만드니까 생각보다 빠르게 완성됐다.

아직 뭘 쓸지 구체적으로 정한 건 아니다. 그냥 공부하다가 정리하고 싶은 게 생기면 올리려고 한다. 거창한 건 없고, 나를 위한 기록이 누군가에게도 도움이 되면 좋겠다는 정도.

일단 시작.
