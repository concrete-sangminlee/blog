트윗 한 줄보다 더 짧게. 이모지만으로 내 연구를 설명할 수 있을까?

## 시도 1: 전체 과정

🏗️ + 🔨 + 🔊 + 💻 + 🤖 = 📄

구조물을 두드려서 소리를 모으고, 컴퓨터에 넣어 AI가 분석하면, 논문이 나온다.

핵심은 전달되는데, 디테일은 완전히 사라졌다.

## 시도 2: 연구의 감정선

💡 → 📚 → 💻 → 😤 → ☕ → 💻 → 😱 → 🐛 → 🔧 → 😅 → 📊 → 📝 → 📧 → 🤞 → ⏳ → 📬 → 😭 or 🎉

아이디어가 떠오르고, 공부하고, 코딩하고, 화내고, 커피 마시고, 또 코딩하고, 버그 발견하고, 고치고, 안도하고, 결과 내고, 논문 쓰고, 투고하고, 기도하고, 기다리고, 결과 받고, 울거나 환호한다.

이게 더 현실적이다.

## 시도 3: 대학원 생활

🌅 → 🚶 → 🏢 → ☕ → 💻💻💻 → 🍜 → 💻💻 → ☕ → 💻 → 🌙 → 🚶 → 🛏️ → 🔁

아침에 일어나서, 연구실 가서, 커피 마시고, 컴퓨터 하고, 밥 먹고, 또 컴퓨터 하고, 커피 또 마시고, 밤에 집 가서, 자고, 반복.

단순하지만 정확하다.

## 결론

이모지로 연구를 설명하는 건 재밌는 실험이다. 핵심을 극한까지 압축하면, 의외로 본질이 드러난다.

다만 이걸 학회 발표에서 쓰면 안 된다.
