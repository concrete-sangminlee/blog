ChatGPT가 코드도 짜고, 글도 쓰고, 수학 문제도 푼다. 그러면 구조역학 문제는 풀 수 있을까?

아직 직접 시도해보지는 않았지만, 경험상 예상되는 결과를 적어본다. 나중에 실제로 해보고 후속 글을 쓸 수도 있겠다.

## 아마 잘할 것 같은 것

**기본 개념 설명**: "전단력이 뭐야?", "모멘트 다이어그램을 어떻게 그려?" 같은 개념 질문은 꽤 잘 설명할 것 같다. 교과서에 있는 내용이니까.

**단순한 문제 풀기**: 단순보에 집중하중이 작용할 때 반력 구하기 같은 기초적인 문제는 맞출 가능성이 높다. 패턴이 정형화되어 있으니까.

## 아마 못할 것 같은 것

**복잡한 구조 해석**: 부정정 구조물이나, 여러 하중 조합이 동시에 작용하는 경우는 어려울 것 같다. 중간 단계에서 실수가 생기면 최종 답이 완전히 달라지는데, AI가 그걸 스스로 검증하기 힘들다.

**물리적 직관**: "이 구조물이 어떤 형태로 변형될까?"를 그림으로 그리는 건 AI가 약한 영역이다. 숫자를 계산하는 것과 물리적 거동을 직관적으로 이해하는 건 다른 문제다.

**단위 실수**: 공학 문제에서는 단위를 잘못 쓰면 답이 크게 틀린다. AI가 단위 환산에서 실수할 가능성이 꽤 있을 것 같다.

## 왜 이게 의미 있나

AI가 공학 문제를 얼마나 풀 수 있는지를 테스트하는 건, AI의 현재 한계를 이해하는 좋은 방법이다. "AI가 다 해줄 텐데 왜 공부하냐"는 질문에 대한 답이 될 수도 있다.

결국 AI는 도구다. 도구가 정확한지 판단하려면, 사용하는 사람이 내용을 알아야 한다.
