sudo. 관리자 권한. 비밀번호. 조심.

## 명령

sudo 앞에 붙임.

## 비밀번호

비밀번호 입력.

## 표시 X

표시 안 됨.

## 권한

관리자.

## 조심

실수 X.

## 결론

sudo 입력은 권한의 무게다. rm -rf는 절대 조심.
