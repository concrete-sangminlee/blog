## 1. f-string 디버깅

Python 3.8부터 `=`를 사용하면 변수명과 값을 동시에 출력할 수 있습니다:

```python
learning_rate = 0.001
batch_size = 32
print(f"{learning_rate=}, {batch_size=}")
# learning_rate=0.001, batch_size=32
```

## 2. dataclass로 config 관리

`@dataclass`를 사용하면 실험 설정을 깔끔하게 관리할 �� 있습니다:

```python
from dataclasses import dataclass, field

@dataclass
class TrainConfig:
    model_name: str = "resnet50"
    learning_rate: float = 3e-4
    batch_size: int = 64
    epochs: int = 100
    tags: list[str] = field(default_factory=list)

config = TrainConfig(learning_rate=1e-3, tags=["experiment-v2"])
```

## 3. `__slots__`로 메모리 절약

대량의 객체를 생성할 때 메모리를 크게 줄일 수 있습니다:

```python
class Point:
    __slots__ = ('x', 'y')

    def __init__(self, x, y):
        self.x = x
        self.y = y
```

## 4. Generator로 메모리 효율적 처리

대용량 데이터를 처리할 때 리스트 대신 generator를 사용하세요:

```python
# Bad: 전체 리스트를 메모리에 올림
results = [process(item) for item in huge_dataset]

# Good: 하나�� 처리
results = (process(item) for item in huge_dataset)
for result in results:
    save(result)
```
