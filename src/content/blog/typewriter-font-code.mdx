## Monaco

맥의 기본. 더 세련됨.

## Consolas

Windows 코드 에디터. Courier보다 읽기 좋다.

## LaTeX

`verbatim` 또는 `lstlisting` 환경. 코드 블록.

## 본문 속 코드

본문 속 짧은 코드. `variable_name`. 인라인 코드.
