발표 직전 파일 점검. 학회장 컴퓨터에서 열어본다. 폰트 깨짐? 이미지 사라짐?

## 폰트

폰트가 없으면 기본 폰트로 대체. 디자인 망가짐.

## PDF로 변환

PDF로 변환하면 폰트 내장. 안전.

## 이미지 경로

이미지 경로가 깨지면 이미지 X. 링크가 아닌 내장.

## 애니메이션

애니메이션이 작동 안 할 수도. 다른 버전.

## 비디오

비디오 재생. 코덱 문제.

## 슬라이드 쇼 모드

슬라이드 쇼 전체 작동. 확인.

## 여러 모니터

듀얼 모니터 설정. Presenter View.

## 백업

USB 백업. 여러 형식.

## 학회 요구

학회가 요구하는 형식. 확인.

## 테스트

스피커 데스크에서 테스트. 스탭 도움.

## 결론

발표 전 파일 점검은 안심의 단계다. 미리 확인하면 당황 없다.
